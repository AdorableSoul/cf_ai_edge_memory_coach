Edge Memory Coach

A simple AI-powered Cloudflare app that chats with the user, remembers notes, and schedules reminders. The backend uses a stateful Durable Object agent with persisted chat history.

Try: Remember that my OS exam is next Tuesday, list my notes, or remind me in 5 minutes to study graphs.

What this app demonstrates
  • LLM with Llama 3.3 on Workers AI
  • Workflow / coordination with a Durable Object agent
  • User input through chat
  • Persistent memory and reminders
